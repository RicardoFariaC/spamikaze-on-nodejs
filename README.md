# Spamikaze on NodeJS

## API Document

#### Return all IPs registered

```zsh
  GET /ip/
```

## Host in your local machine

Go inside the project directory

```zsh
  cd spamikaze-back
```

Install all dependencies from package.json

```zsh
  npm install
```

It will compile every .ts file and start the server

```zsh
  npm run start:dev
```
